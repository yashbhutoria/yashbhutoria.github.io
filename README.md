## Design Philosophy

This resume adopts a cyberpunk/hacker aesthetic reminiscent of movies like The Matrix, featuring:
- Black background with various shades of green text
- Monospace font (Courier New) for authentic terminal feel
- Terminal window header with colored dots
- Glowing text effects and smooth animations
- Matrix digital rain background animation

## Technical Implementation

- **Pure HTML/CSS/JavaScript** - No external dependencies
- **CSS Grid & Flexbox** - Modern responsive layout
- **Canvas API** - Matrix rain animation effect
- **CSS Animations** - Typing effects and glowing text
- **Mobile-first design** - Optimized for all screen sizes

## Local Development

Simply open `index.html` in any modern web browser. No build process or dependencies required.
